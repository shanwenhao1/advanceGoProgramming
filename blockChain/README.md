# 区块链

 简单初步概括: 区块链是通过密码学的方式形成的一个由集体维护的分布式数据库. 
 实际上是分布式的点对点的网络
 
区块链的特点:
- 去中心化
- 安全性: 分布式系统, 可靠性高
- 去信任: 一切社会行为都要建立在"信任"的基础上



## 所需理论基础
- 密码学
- 共识算法
- 博弈论
